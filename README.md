# School Project - Login System

## 📌 Description
A simple login system built using HTML, CSS, and JavaScript.

It demonstrates basic frontend development skills including structure, styling, and simple interactivity like login, welcome screen, and logout functionality.

## 📁 Project Structure
- `index.html` → Main login page
- `css/style.css` → Styling and design
- `js/script.js` → Functionality and logic

## 🚀 Features
- Login form interface
- Basic form validation
- Welcome screen after login
- Logout functionality
- Simple and clean UI

## 🎯 Purpose
This project is built to practice frontend development and understand how to organize a project using GitHub properly.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
